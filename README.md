# GenAI 기초 2: 멀티모달 콘텐츠 제작  
## 마음노트 AI 기반 브랜드 광고 패키지

## 1. 프로젝트 개요

이 프로젝트는 GenAI 기초 2: 멀티모달 콘텐츠 제작 과제로 진행한 AI 기반 브랜드 광고 패키지입니다.

가상의 브랜드 **마음노트**를 설정하고, 코디세이 학습 네이트의 이미지 생성 AI, 영상 생성 AI, 음성 생성 AI를 활용하여 10초 이내의 브랜드 광고 영상을 제작했습니다.

최종 결과물은 다음 두 가지입니다.

- 스토리보드/기획 문서 PDF
- 10초 이내 브랜드 광고 영상 MP4

## 2. 브랜드 아이덴티티

- 브랜드명: 마음노트
- 브랜드 유형: AI 기반 감정 기록 서비스
- 타깃: 불안, 무기력, 외로움, 고립감을 느끼지만 상담이나 도움 요청은 아직 부담스러운 20~30대 청년
- 톤앤매너: 차분함, 따뜻함, 과장 없는 위로, 미니멀한 감성
- USP: 감정을 진단하거나 평가하지 않고, 짧은 기록을 통해 자기 마음을 정리하는 첫 단계를 제공
- 핵심 메시지: 말하지 못한 마음도, 기록하면 돌봄의 시작이 된다.
- CTA: 오늘의 마음, 한 줄부터.

## 3. 광고 콘셉트

광고는 **문제 제시 → 전환 → 해결/제안** 구조로 구성했습니다.

| 구간 | 내용 | 역할 |
|---|---|---|
| 씬 1 | 밤거리에서 지친 채 혼자 걷는 여성 | 문제 제시 |
| 씬 2 | 늦은 밤 스마트폰으로 마음을 기록하는 장면 | 전환 |
| 씬 3 | 아침 햇살이 들어오는 책상 위 브랜드 메시지 | 해결/제안 |

최종 영상은 밤거리의 고립감, 실내 기록 장면, 아침 햇살의 엔드카드로 이어지며 마음노트의 브랜드 메시지를 전달합니다.

## 4. 사용 도구

### 이미지 생성 AI

- 사용 도구: 코디세이 학습 네이트 > 미디어 > 이미지
- 사용 목적:
  - 마음 기록 장면 생성
  - 브랜드 엔드카드 배경 생성
  - 16:9 화면 비율에 맞는 시각 자료 제작

### 영상 생성 AI

- 사용 도구: 코디세이 학습 네이트 > 미디어 > 영상
- 사용 목적:
  - 밤거리에서 단발머리 여성이 지친 모습으로 걷는 모션 컷 생성

### 음성 생성 AI

- 사용 도구: 코디세이 학습 네이트 > 미디어 > 음성
- 사용 목적:
  - 광고 전체에 들어갈 내레이션 생성
  - 여러 목소리를 비교한 뒤 가장 차분하고 자연스러운 음성 선택

### 통합 편집 도구

- 사용 도구: CapCut
- 사용 목적:
  - 영상, 이미지, 음성 파일 통합
  - 자막 삽입
  - 오디오 조정
  - 최종 MP4 내보내기

## 5. 최종 결과물 정보

- 파일명: `maumnote_ad_video.mp4.mp4`
- 길이: 약 9.8초
- 해상도: 1920×1080
- 화면 비율: 16:9
- 프레임레이트: 30fps
- 비디오 코덱: H.264
- 오디오 코덱: AAC
- 구성: 영상 생성 컷 1개 + 이미지 생성 컷 2개 + AI 생성 내레이션

## 6. 주요 프롬프트 수정 기록

### 씬 1 수정

초기에는 실내 책상 앞에서 스마트폰을 바라보는 장면을 생성했으나, 씬 2와 구성이 너무 비슷해 문제 제시 장면으로서 차별성이 부족했습니다.  
이후 밤거리에서 지친 채 걷는 단발머리 20대 한국 여성으로 방향을 수정하여 문제 제시와 전환 구조를 명확히 했습니다.

### 씬 3 수정

처음 생성한 엔드카드 배경은 왼쪽은 밤, 오른쪽은 낮처럼 보여 시각적 일관성이 떨어졌습니다.  
이후 전체 장면을 새벽 아침 햇살이 들어오는 따뜻한 책상 장면으로 통일했습니다.

### 음성 생성 수정

여러 TTS 목소리를 비교했으며, 일부 음성은 특정 표현의 억양이 어색했습니다.  
최종적으로 브랜드의 차분하고 따뜻한 톤에 가장 적합한 음성을 선택했습니다.

## 7. 제작 과정에서의 문제와 대응

- 이미지 생성 중 timeout 오류 발생 → 모델/화질 옵션 조정 및 프롬프트 간결화
- 캐릭터 일관성 문제 발생 → “20대 한국 여성, 단발머리, 뒤 또는 측면 구도”를 명시
- 영상과 이미지 간 의상 차이 발생 → 인물의 핵심 요소와 정서적 톤의 일관성을 우선
- 편집 툴 워터마크 문제 발생 → 최종 편집은 CapCut으로 진행
- 영상 생성 결과의 배경음악 문제 → 최종본에서는 내레이션 중심으로 오디오 정리

## 8. 제작 파이프라인 및 제약 고려

이번 프로젝트는 기획 의도 → 프롬프트 설계 → 이미지 생성 → 영상 생성 → 음성 생성 → 통합 편집 → 최종 검수의 순서로 진행했습니다.

### 제작 파이프라인

1. 브랜드 기획  
   - 마음노트라는 가상의 AI 감정 기록 서비스를 설정했습니다.
   - 타깃, 톤앤매너, USP, 핵심 메시지를 먼저 정의했습니다.

2. 스토리보드 설계  
   - 광고를 문제 제시 → 전환 → 해결/제안 구조로 구성했습니다.
   - 씬 1은 밤거리에서 지친 채 걷는 장면, 씬 2는 스마트폰으로 마음을 기록하는 장면, 씬 3은 브랜드 메시지를 전달하는 엔드카드로 설계했습니다.

3. 이미지 생성  
   - 코디세이 학습 네이트의 이미지 생성 AI를 사용해 씬 2 기록 장면과 씬 3 엔드카드 배경을 생성했습니다.
   - 처음에는 정사각형 이미지가 생성되어 16:9 영상 편집 시 검은 여백과 크롭 문제가 발생할 수 있었습니다.
   - 이후 16:9 비율에 맞는 이미지를 다시 생성하여 최종 영상에 적합하도록 조정했습니다.

4. 영상 생성  
   - 코디세이 학습 네이트의 영상 생성 AI를 사용해 씬 1의 밤거리 걷기 모션 컷을 생성했습니다.
   - 전체 광고를 영상 생성 AI로 한 번에 만들지 않고, 씬 1 모션 컷만 생성한 뒤 이미지와 음성을 편집 도구에서 결합했습니다.
   - 이 방식은 캐릭터와 장면을 통제하기 쉽고, 생성 실패 시 수정 범위를 줄일 수 있기 때문에 선택했습니다.

5. 음성 생성  
   - 코디세이 학습 네이트의 음성 생성 AI를 사용해 내레이션을 생성했습니다.
   - 여러 목소리와 문장을 비교한 뒤, 브랜드의 차분하고 따뜻한 톤에 가장 적합한 음성을 선택했습니다.
   - 10초 이내 영상 조건을 맞추기 위해 문장을 짧게 조정했습니다.

6. 통합 편집  
   - CapCut을 사용해 영상, 이미지, 내레이션, 자막을 하나의 MP4 파일로 통합했습니다.
   - 영상 생성 결과에 포함된 배경음악은 내레이션 전달을 방해할 수 있어 제거하거나 낮추는 방향으로 조정했습니다.
   - 최종 영상은 1920×1080, 16:9, 30fps, 약 9.8초로 내보냈습니다.

### 크레딧 및 대기열 제약 고려

- 영상 생성은 이미지나 음성 생성보다 크레딧 소모가 크고 실패 가능성이 높기 때문에, 전체 광고를 한 번에 영상으로 생성하지 않았습니다.
- 먼저 이미지와 스토리보드를 확정한 뒤, 필요한 부분만 영상 생성 AI로 제작하여 크레딧 낭비를 줄였습니다.
- 이미지 생성 단계에서 timeout 오류가 발생했기 때문에, 프롬프트를 간결하게 조정하고 생성 옵션을 변경하여 재시도했습니다.
- 음성 생성은 상대적으로 크레딧 소모가 적은 모델부터 테스트하고, 여러 목소리를 비교한 뒤 최종 음성을 선택했습니다.
- 편집 도구는 워터마크 문제가 없는 CapCut을 사용하여 최종 제출물의 완성도를 유지했습니다.

### 최종 파이프라인 요약

브랜드 기획 → 씬별 스토리보드 작성 → 이미지 생성 → 영상 생성 → 음성 생성 → CapCut 통합 편집 → 최종 MP4 내보내기 → GitHub 업로드

## 10. 최종 결론

이번 작업은 단순히 AI로 이미지를 생성하는 것이 아니라, 브랜드 기획 의도에서 출발해 이미지, 영상, 음성 생성 결과물을 하나의 광고 영상으로 통합하는 제작 파이프라인을 설계하는 과정이었습니다.

마음노트 광고는 감정을 진단하거나 과장된 위로를 제공하는 방식이 아니라, “한 줄 기록”이라는 낮은 진입장벽을 통해 자기 마음을 정리하는 시작점을 제안합니다.

최종 영상은 “말하지 못한 마음도, 기록하면 돌봄의 시작이 된다”는 브랜드 메시지를 전달합니다.
