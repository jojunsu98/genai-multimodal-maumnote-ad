## 17. 동일한 스토리보드를 다른 도구로 제작할 경우의 변화 예측

| 제작 단계  | 다른 도구 사용 시 예상 변화                                                             | 종합 판단                                                                 |
| ------ | ---------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| 이미지 생성 | Midjourney, DALL·E, Imagen 계열 도구를 사용하면 조명, 색감, 인물 표현과 공간 연출 방식이 달라질 수 있음     | 이미지의 미적 완성도는 향상될 수 있지만, 동일 인물과 소품의 일관성을 유지하려면 프롬프트 관리와 반복 생성이 추가로 필요함 |
| 영상 생성  | Veo, Kling, Seedance 등 다른 영상 생성 도구를 사용하면 카메라 움직임, 인물 모션과 장면 전환의 특성이 달라질 수 있음 | 고급 모델일수록 인물 유지와 모션 품질이 향상될 가능성이 있지만, 크레딧과 비용, 생성 대기시간에 대한 부담이 커질 수 있음 |
| 음성 생성  | 고급 TTS 도구를 사용하면 억양, 호흡과 감정 표현이 더욱 자연스러워질 수 있음                                | 10초 내외의 짧은 광고에서는 음성 모델의 성능 차이보다 문장 길이, 발음 정확도와 브랜드 톤의 일치가 더 중요함       |
| 통합 편집  | Premiere Pro나 DaVinci Resolve를 사용하면 색보정, 장면 전환과 오디오 믹싱을 더욱 정밀하게 조정할 수 있음     | 이번 과제에서는 CapCut만으로도 컷 편집, 자막 삽입, 음량 조정과 최종 출력에 필요한 기능을 충분히 구현할 수 있었음  |
| 텍스트 합성 | 영상 생성 AI를 통해 스마트폰 화면의 한글 문구나 브랜드 로고를 직접 생성할 수 있음                             | 한글이 깨지거나 글자 형태가 변형될 가능성이 높으므로, 편집 프로그램에서 텍스트와 로고를 별도로 합성하는 방식이 더 안정적임 |

같은 스토리보드를 사용하더라도 어떤 도구를 선택하느냐에 따라 이미지의 미감, 인물의 움직임, 카메라 연출과 영상의 전체적인 분위기는 달라질 수 있습니다.

그러나 더 고급인 모델이나 유료 도구를 사용한다고 해서 반드시 최종 결과물의 완성도가 높아지는 것은 아닙니다. 도구를 선택할 때는 다음 요소를 함께 고려해야 합니다.

* 인물과 공간의 일관성
* 프레임 간 형태 안정성
* 한글 문구와 로고의 정확도
* 크레딧과 이용 비용
* 생성 대기시간
* 결과물의 수정 가능성
* 최종 편집 도구와의 호환성

이번 프로젝트에서는 최고 수준의 생성 품질만을 추구하기보다, 결과물의 수정 가능성과 제어 가능성, 크레딧 효율을 종합적으로 고려하여 도구를 선택했습니다.

## 18. 60초 광고를 15초로 축약할 경우의 전략

### 18.1. 60초 버전 구성 가정

* **0~10초:** 밤거리에서 지친 일상과 문제 상황 제시
* **10~25초:** 말하지 못한 감정이 쌓이는 상황 설명
* **25~40초:** 스마트폰을 통해 자신의 감정을 기록하는 장면
* **40~50초:** 기록 이후 긴장이 완화되는 과정
* **50~60초:** 브랜드명, 슬로건과 CTA 제시

### 18.2. 15초 축약 구성

| 구간     | 주요 내용               | 편집 전략                                     |
| ------ | ------------------- | ----------------------------------------- |
| 0~4초   | 밤거리에서 지친 채 혼자 걷는 장면 | 여러 문제 상황을 나열하지 않고 핵심 감정을 대표하는 한 장면으로 압축   |
| 4~9초   | 스마트폰에 감정을 기록하는 장면   | 기록 행위, 화면 내용과 내레이션을 동시에 제시하여 정보 전달 시간을 단축 |
| 9~13초  | 새벽 침실에서 편안하게 잠든 장면  | 감정 기록 이후 긴장이 완화되고 휴식으로 이어지는 결과를 시각적으로 표현  |
| 13~15초 | 브랜드명과 CTA           | 브랜드명과 핵심 행동 제안만 간결하게 노출                   |

### 18.3. 축약 기준

1. 반복되는 문제 상황과 유사한 장면은 제거합니다.
2. 긴 설명이나 인터뷰는 짧은 내레이션과 자막으로 대체합니다.
3. 각 장면에는 하나의 명확한 역할만 부여합니다.
4. `문제 제시 → 감정 기록 → 휴식 → CTA`의 핵심 흐름은 유지합니다.
5. 브랜드명과 CTA를 인지할 수 있도록 마지막 노출 시간을 확보합니다.
6. 내레이션은 세 문장 이하로 압축합니다.
7. 시각적 정보와 음성 정보를 중복해서 설명하지 않고 서로 보완하도록 구성합니다.

### 18.4. 15초 버전 내레이션 예시

> 말하지 못한 마음이 쌓이는 밤.
> 한 줄 적으면, 마음이 조금 보입니다.
> 마음노트. 오늘의 마음, 한 줄부터.

이러한 방식은 60초 버전의 구체적인 설명과 상황 묘사를 줄이는 대신, 마음노트가 전달하려는 핵심 메시지와 정서적 흐름을 15초 안에서 유지할 수 있도록 합니다.

## 19. 최종 학습 내용

이번 프로젝트의 제작과 수정 과정을 통해 확인한 핵심 학습 내용은 다음과 같습니다.

1. 정지 이미지를 영상 타임라인에 배치하는 것만으로는 영상적 완성도가 확보되지 않습니다.
2. 이미지의 개별 품질이 높더라도 움직임과 장면 전환이 부족하면 PPT나 슬라이드처럼 보일 수 있습니다.
3. 장면별 밝기, 색온도와 시간대가 다르면 각 장면이 하나의 영상처럼 자연스럽게 연결되지 않습니다.
4. 배경음악은 단순한 장식이 아니라 장면 사이의 감정 흐름을 연결하는 멀티모달 요소입니다.
5. 생성형 영상에서는 얼굴, 손, 머리카락, 침구와 소품의 프레임 간 일관성을 반드시 검수해야 합니다.
6. 프롬프트가 길고 상세하다고 해서 반드시 안정적인 결과가 생성되는 것은 아닙니다.
7. 호흡, 커튼, 조명, 카메라와 소품의 움직임을 동시에 요구하면 생성 결과가 불안정해질 수 있습니다.
8. 움직임이 거의 필요하지 않은 장면에서는 AI 영상 생성보다 정지 이미지와 편집 효과를 활용하는 편이 더 안정적일 수 있습니다.
9. 한글 문구와 브랜드 로고는 생성 AI에 직접 맡기기보다 편집 도구에서 별도로 합성하는 편이 정확합니다.
10. 영상 생성 결과는 전체 구간을 그대로 사용하기보다 형태가 안정적인 구간을 선별하여 활용할 수 있습니다.
11. 피드백은 단순한 평가가 아니라 장면 구조, 오디오, 색감과 메시지를 다시 설계하는 기준이 될 수 있습니다.
12. 멀티모달 콘텐츠 제작은 이미지, 영상과 음성을 각각 만드는 작업이 아니라 서로 다른 요소를 하나의 흐름으로 통합하는 작업입니다.
13. 생성형 AI 활용 역량에는 프롬프트 작성뿐 아니라 실패 결과를 분석하고 적절한 대안을 선택하는 판단 능력도 포함됩니다.
14. 더 많은 움직임이나 더 비싼 모델이 항상 더 높은 완성도를 의미하는 것은 아닙니다.
15. 최종 검수에서는 생성 자체의 성공 여부보다 브랜드의 기획 의도와 시청자의 경험을 기준으로 결과물을 판단해야 합니다.

## 20. 최종 결론

이번 프로젝트는 단순히 AI를 활용하여 이미지나 영상을 생성하는 작업이 아니라, 브랜드의 기획 의도에서 출발하여 이미지, 영상, 음성, 자막과 배경음악을 하나의 광고 흐름으로 통합하는 과정이었습니다.

초기 완성본은 밤거리 영상과 두 장의 정지 이미지를 결합하여 메시지를 전달하는 방식으로 제작되었습니다. 그러나 결과물을 검토하는 과정에서 다음과 같은 한계가 확인되었습니다.

* 일부 장면이 영상보다 PPT나 슬라이드처럼 보임
* 배경음악이 없어 정서적 분위기가 충분히 형성되지 않음
* 마지막 장면의 밝기가 전체 영상의 톤에서 벗어남
* 어두운 밤에서 밝은 아침으로 이어지는 변화가 지나치게 급격함
* 이미지 기반 영상에서 인물과 침구의 형태가 프레임마다 흔들림

이러한 문제를 보완하기 위해 다음과 같은 수정 작업을 진행했습니다.

* 스마트폰 기록 장면에 자연스러운 움직임 추가
* 밝은 아침 엔드카드를 어두운 새벽 침실 장면으로 교체
* 잠든 여성의 단발머리와 자연스러운 앞머리 형태 보완
* 사용을 마친 휴대폰이 침대 옆 탁자 위에 놓인 상태로 표현
* 새벽 3~4시의 어두운 청색광과 약한 간접조명 적용
* 영상의 정서적 흐름을 연결하는 잔잔한 배경음악 추가
* 이미지 투 비디오 결과에서 얼굴, 손, 침구와 소품의 변형 여부 검수
* 영상 생성 결과가 불안정할 경우 형태가 안정적인 구간만 선별하여 활용
* 필요한 경우 AI 영상 대신 정지 이미지와 미세한 편집 모션으로 대체

최종 마음노트 광고는 다음과 같은 정서적 흐름을 전달합니다.

> 말하지 못한 마음이 쌓이는 밤
> → 짧은 기록을 통해 자신의 감정을 확인하는 순간
> → 모든 문제가 해결된 것은 아니지만 잠시 긴장을 내려놓고 쉴 수 있는 상태

마음노트는 사용자의 감정을 진단하거나 모든 문제를 즉시 해결하는 서비스가 아닙니다. 한 줄 기록이라는 낮은 진입장벽을 통해 자신의 감정을 확인하고, 자기 돌봄을 시작할 수 있는 작은 행동을 제안하는 서비스입니다.

이번 수정 과정은 생성형 AI 도구를 사용하는 기술뿐 아니라, 실제 피드백을 분석하고 결과물의 문제를 발견한 뒤 기획, 프롬프트, 이미지, 영상과 오디오를 다시 조정하는 콘텐츠 개선 역량을 보여주는 과정이었습니다.

최종적으로 이 프로젝트를 통해 확인한 가장 중요한 점은 다음과 같습니다.

> 좋은 멀티모달 콘텐츠는 생성 결과를 많이 사용하는 콘텐츠가 아니라, 각각의 요소를 필요한 만큼 선택하고 조정하여 하나의 일관된 메시지로 연결한 콘텐츠입니다.
